# 266. Palindrome Permutation

Given a string s, return true if a permutation of the string could form a palindrome
and false otherwise.

`1` Example 1:

```
Input: s = "code"
Output: false
```

`2` Example 2:

```
Input: s = "aab"
Output: true
```

`3` Example 3:

```
Input: s = "carerac"
Output: true
```

Constraints:

```
1 <= s.length <= 5000
s consists of only lowercase English letters.


```
